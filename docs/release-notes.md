## Release Notes v2.9.0

### New Features

* **Shortcut Search Separation** Added a differentiated shortcut search mechanism for text-based and key-based queries.
* **File and Folder Copy Support** Added support for copying single files, single folders, and recursive folder contents.

### Fixes

* **Character Count Fix** Fixed incorrect character counting behavior.
* **Shortcut Handling Fixes** Improved shortcut management and related behavior.

### Improvements

* **Translations Updated** Added and improved translation entries.
* **General Shortcut Improvements** Refined shortcut interactions and usability.
